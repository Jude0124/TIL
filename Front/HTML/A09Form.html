Form

사용자로부터 어떠한 값을 받을 목적으로 기술 전달될 데이터는 form 태그에 포함되어야 한다.
GET : 256Byte ~ 4096Byte 이내의 크기의 데이터만 전송 가능.
주소줄에 address?key=value&key=value......
POST: 크기 상관없음.
body 처음에 붙어서 넘어간다.(내용을 볼 수 없음)

상품 주문
size는 필드의 크기(입력할 수 있는 칸의 크기)
maxlength 정의한 필드에 입력할 수 있는 크기의 최대 값. 10이면 10자만 입력 가능. size 가 20이면 크기는 20자만 입력이 가능한데 입력 할 수 있는 최대 값은 10자만 입력 가능
required : 값이 반드시 입력해야한다는 것을 의미한다. 이름이 비어있을 수 없음, 값이 반드시 전송되지않는다.
readonly : 읽기전용, 사용자가 값을 입력할 수 없음
disabled : 사용하지않음. 사용자가 값을 입력할 수 없음, 지정한 action에 값을 전달하지 않음
type을 hidden으로 주면 해당 필드는 사용자 몰래 값을 전송 받을 수 있다. 화면에서 보이지는 않는다.

이 히든 필드는 화면에 표시되지 않음.
